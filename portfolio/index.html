Игра Raindrops

Приложение - тренажёр по математике, позволяющий закрепить навыки арифметических действий. В левой части игрового поля, оформленного как морской пейзаж, падают капли дождя. Внутри каждой капли - математическое выражение. Если игрок правильно решил математическое выражение и успел ввести ответ до того, как капля упала в море, его счёт увеличивается. Использованы HTML5, CSS3, JavaScript. В процессе игры присутсвует анимация и звук.

Прогноз погоды fancy-weather

Когда пользователь открывает приложение, все данные на странице относятся к текущему местоположению пользователя. В строке поиска осуществляется поиск по населённому пункту. Фоновое изображение изменяется при обновлении содержания страницы или при клике по кнопке для переключения фонового изображения в блоке контроля. Использованы HTML5, CSS3, JavaScript. Встроена карта Google, данные с сервера приходят при помощи асинхронных запросов.

Верстка simple-singolo

В ходе обучения на курсе up-skill-lab выполнено задание simple-singolo. Необходимо было сверстать страницу согласно макету. Использовать JavaScript для создания адаптивного меню, слайдера, переключения табов. Макет сверстан для 4х размеров экрана, применялся Flexbox, медиа-запросы.
